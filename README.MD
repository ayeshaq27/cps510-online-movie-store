Go to the following Github link: https://github.com/ayeshaq27/cps510-online-movie-store

In the Github link, click the “Download ZIP” button that is located after clicking the Code button.

After installing the .zip file, extract the files into a directory that is readable by a compiler (such as Visual Studio Code).

In the terminal, type and enter “pip install oracledb”.

In the directory that the files are extracted in, run the “landing_page.py” file. This will send you to the login screen, in which instructions with that screen are shown below.
